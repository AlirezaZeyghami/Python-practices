# Employee Information:
```Text
A database containing information about the name, height, and weight of employees in a company has been designed. 
Write a program that reads the information from this database file and prints the employees in order of height, along with their name and weight. 
(If two employees have the same height, print the one with the lower weight first.) 
It is guaranteed that no two individuals have both the same height and the same weight.
The format of the database table information is as follows:
Sample input: 
(The information below is defined in the database, and this is a sample; the number of individuals may vary)

Height | Weight | Name
180 | 75 | Amin
190 | 90 | Mahdi
175 | 75 | Mohammad
175 | 60 | Ahmad
```
