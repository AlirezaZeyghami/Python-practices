# Save login information:
```Text
Write a program that takes an email and password as input and stores them in a database under the fields username and password. 
Submit only the code file. The written code must be capable of taking input information and storing and recording the information in a database file of your choice.
It is mandatory to validate the email format as input. 
If the input format is not in the correct email format, show the user a correct example and prompt for input again. 
The password must include both numbers and letters.

The correct email pattern consists of two strings and one expression (containing letters and/or numbers and letters combined), separated by @ and ., as a valid email. It should be in the following form:
expression@string.string
```
